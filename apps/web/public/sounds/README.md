# System Sound Files

Place your MP3 sound files in this directory:

| File Name | Description | Recommended Duration |
|-----------|-------------|---------------------|
| `notification.mp3` | Default notification sound | 0.5-1s |
| `complete.mp3` | Task/quest completion | 1-2s |
| `levelup.mp3` | Level up celebration | 2-3s |
| `warning.mp3` | Warning/alert sound | 0.5-1s |
| `failure.mp3` | Failure/penalty sound | 1-2s |

## Recommended Sound Style

For the Solo Leveling aesthetic, look for:
- Futuristic UI sounds
- Sci-fi notification tones
- Digital/holographic beeps
- Deep bass hits for important events

## Free Sound Resources

- [Freesound.org](https://freesound.org) - Search "sci-fi notification", "game UI"
- [Mixkit](https://mixkit.co/free-sound-effects/) - Free game sounds
- [Zapsplat](https://www.zapsplat.com/) - UI sound effects

## Sound Settings

Users can control sounds via:
- `isMuted()` - Check mute state
- `setMuted(true/false)` - Set mute preference
- `setGlobalVolume(0-1)` - Set volume level

These preferences are saved to localStorage.
